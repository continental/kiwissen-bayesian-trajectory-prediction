Code StructureÔÉÅ

Models are defined by a ModelFactory, which is an abstract base class. Realizations usually consist of an base model (like Covernet) and a probabilistic modeling Mixin (like VIMixin), resulting in a complete ModelFactory (CovernetVI).

Executing a model uses the Trainer, which creates metrics, runs the training and the evaluation. TuneableTrainer is a wrapper for use with ray tune (see program arguments).
