bayes_covernet.tf.model.training.Trainer

class Trainer(model_factory, dir, current_time=None, name='model_0', start_epoch=0, epochs=20, batch_size=64, load_model=None, resume_model=None, is_tune=False, multi_label=False, multitask_lambda=0.0, mixed_precision=True, early_stop_delay=300, early_stop_patience=100, **kwargs)[source]

Bases: object

Basic trainer class. Handles model training, loading and runtime logging.

Parameters

  • model_factory – A model factory class, derived from :class:Covernet

  • dir (Path) – Root folder for the experiment

  • str,None – Current time string, if None, defaults to now

  • name (str) – Model name, usually model_i, with i as the index

  • start_epoch (int) – Number of the first epoch

  • epochs (int) – Number of total epochs

  • batch_size (int) – Number of examples per batch

  • load_model (Path) – Model file path to load (as initialization)

  • resume_model (Path) – Model file path to load (as resume)

  • is_tune (bool) – If True, several logging methods are disabled (handled via ray)

  • multi_label (bool) – If True, a multi label loss is used (BCE instead of CE)

  • multitask_lambda (float) – If > 0, multitask output is assumed

  • mixed_precision (bool) – If True, mixed float16 is used.

  • kwargs (dict) – Additional hyperparameters, passed to the model factory.
