bayes_covernet.tf.model.covernet_utils.multipath_to_trajs

multipath_to_trajs(y_true, y_preds, lattice)[source]

Maps the MultiPath output to the trajectory represented by that class, with shift. Only applied to y_pred

Parameters

  • y_true (tf.Tensor) – ground truth, not used

  • y_pred (tf.Tensor) – predictive class probabilities

  • lattice (tf.Tensor) – array of trajectories for each class with (CLASS,STEP,POS) format

Returns

y_true, array with (CLASS,STEP,POS), ordered by the prediction probability

Return type

tf.Tensor, tf.Tensor
