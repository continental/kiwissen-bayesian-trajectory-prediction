update_state(y_true, y_pred, sample_weight=None)[source]

Accumulates metric statistics.

y_true and y_pred should have the same shape. Args:

y_true: The ground truth values. y_pred: The predicted values. sample_weight: Optional weighting of each example. Defaults to 1.

Can be a Tensor whose rank is either 0, or the same rank as y_true, and must be broadcastable to y_true.

Returns:

Update op.
