bayes_covernet.tf.metrics.class_nll

class_nll(y_true, y_pred, cls_idx=0, mapper=None)[source]

Computes the negative log likelihood for observing any label with cls_idx.

Parameters

  • y_true (tf.Tensor) – Groundtruth

  • y_pred (tf.Tensor) – Predictions

  • cls_idx (int) – 0 or 1, for active or inactive labels

  • mapper (Callable,None) – Function that projects y_true, y_pred into coordinate space.

Returns

Metric value per instance

Return type

tf.Tensor
