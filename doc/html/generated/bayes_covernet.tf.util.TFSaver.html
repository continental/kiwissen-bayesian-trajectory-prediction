bayes_covernet.tf.util.TFSaver

class TFSaver(path, model, optimizer, measures=['epoch'], is_min=[False], last_n=5)[source]

Bases: keras.callbacks.Callback

Model saver using tensorflow checkpoint, instead of model save. Supports multiple metrics and keeps the last_n models per measure.

Parameters

  • path (Path) – root checkpoint store path

  • model (Model) – Tensorflow model

  • optimier (Optimizer) – Keras optimizer

  • measures (list[str]) – Measures to watch. Can contain epoch

  • is_min (list[bool]) – True, if save on lower measure value. Must have same length as measures.

  • last_n (int) – How many models to keep (per measure).

on_epoch_end(epoch, logs=None)[source]

Called at the end of an epoch.

Subclasses should override for any actions to run. This function should only be called during TRAIN mode.

Args:

epoch: Integer, index of epoch. logs: Dict, metric results for this training epoch, and for the

validation epoch if validation is performed. Validation result keys are prefixed with val_. For training epoch, the values of the

Model’s metrics are returned. Example`{‘loss’: 0.2, ‘accuracy’:

0.7}`.
