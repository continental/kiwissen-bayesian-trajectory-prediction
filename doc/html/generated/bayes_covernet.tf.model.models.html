class CovernetVI(dataset_size, batch_size, lr=5e-05, momentum=0.9, max_lr=0.002, eps_set=4, posterior_temp=None, gvcl_lambda=100.0, prior_stddev=1.0, stddev_mean_init=0.005, warmup_epochs=0, anneal_epochs=0, lr_warmup_epochs=0, lr_anneal_epochs=0, lr_decay_epochs=0, train_mc_samples=1, multi_label=False, freeze_mean=False, cycle=False, cycle_power=1.0, use_renyi=False, tied_mean=False, prior_model=False, clip_norm=0.0)[source]

Bases: bayes_covernet.tf.model.resnet_models.ResNet50VIMixin, bayes_covernet.tf.model.models.Covernet

Variational Inference CoverNet model

Parameters

  • dataset_size (int) – Number of examples in the dataset

  • batch_size (int) – Number of example sper batch

  • lr (float) – learning rate

  • momentum (float) – momentum

  • eps_set (int) – Epsilon of the Covernet trajectory set (2,4,8)

  • posterior_temp (float,None) – Temperature of the posterior (<=1.0), None results in tempering by 1/batch_size. Identical to GVCL beta.

  • gvcl_lambda (float) – Lambda constant of GVCL, requires a prior_model

  • prior_stddev (float) – Stddev of the prior

  • stddev_mean_init (float) – Average initialization value for the posterior stddev

  • warmup_epochs (int) – Number of epochs without KL regularizer

  • anneal_epochs (int) – Number of epochs to scale the KL weight from 0 to 1

  • train_mc_samples (int) – Number of Monte Carlo samples during training

  • multi_label (bool) – If true, assumes multiple, valid trajectories per example

  • freeze_mean (bool, str) – If true, initial parameter mean is frozen. ‘adaptive’ freezes the mean if tempered KL > nll

  • use_renyi (bool) – Use Renyi alpha divergence instead of KL

  • tied_mean (bool) – If true, the prior mean is always identical to the posterior mean. Only valid for prior_model=False (default)

  • prior_model (str) – Checkpoint name of the prior model. In case of a CovernetDet model, weights are loaded as prior mean (continual learning mode).
